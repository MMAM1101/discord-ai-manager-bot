# 🤖 بوت ديسكورد المبرمج ذاتياً بالذكاء الاصطناعي (Gemini Core)

تم إنتاج ورفع هذا الكود بالكامل وبشكل ذكي ومؤتمت.

## 🚀 طريقة التشغيل السريع والمجاني لمدّة 24 ساعة عبر Railway:

1. توجّه إلى موقع [Railway.app](https://railway.app).
2. أنشئ حساباً وقم بربطه بحسابك على GitHub.
3. انقر على **New Project** ثم اختر **Deploy from GitHub repository**.
4. حدد مستودع هذا البوت (`discord-ai-manager-bot`).
5. اذهب لتبويب المتغيرات الإعداديّة **Variables** وأدخل القيمتين الضروريتين بنجاح ليعمل البوت:
   - `DISCORD_BOT_TOKEN` : توكن البوت الحصري من صفحة المطورين بديسكورد.
   - `GEMINI_API_KEY` : مفتاح الذكاء الاصطناعي لجوجل من منصة Google AI Studio.

6. انطلق! البوت الخاص بك سيعمل ذاتياً ويبرمج نفسه ويتفاعل بنجاح وبسرعة خارقة في الذاكرة السحابية الحية! 🎉
